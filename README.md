# userpanel (react, ant.design)

## Getting Started  
Open [http://localhost:3000/](http://localhost:3000/) to view it in the browser.

     
## Prerequisites    
- Node.js
- npm

## Installing

#### Node.js
Download the node.js source code or a pre-Built installer for Your platform

#### npm
`npm install`  

#### ant design 
`npm install antd`

## Business Page 

* login user with (OTP)
* view users list
* update users
